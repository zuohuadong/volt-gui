# Reasonix v2

> DeepSeek-native coding agent — ground-up rewrite.

This branch is a clean-slate rebuild:

- **Kernel rewritten in Go** — the agent loop, tooling, and runtime move from TypeScript to Go.
- **Web removed** — no web client in v2.
- **Desktop client redone** — rebuilt from scratch.

## Status

🚧 Early work in progress. The architecture and layout are still taking shape.

The previous TypeScript implementation (CLI + web + desktop) remains available on the
[`main`](https://github.com/esengine/DeepSeek-Reasonix/tree/main) branch for reference.
