# Mpx 开发指南

本 Skill 提供了 Mpx 框架的基础开发指南索引。Mpx 是一款致力于提高小程序开发体验和开发效率的增强型小程序框架。

## 使用方式

本 skill 只保留 Mpx 任务的触发和检查框架，不再引用机器本地绝对路径。处理 Mpx 任务时：

1. 先检查目标项目是否自带 Mpx 文档、示例或现有组件模式。
2. 若项目没有本地文档，再查 Mpx 官方文档或依赖包里的类型/示例。
3. 修改 `.mpx` 文件前先识别输出平台：小程序、Web、React Native 或多端。
4. 保持现有目录结构、条件编译风格、样式单位和数据组织方式。
5. 涉及 React Native 样式兼容时，同时加载 `mpx-rn-style-guide`。

## 重点检查项

- App / Page / Component 生命周期是否符合项目既有写法
- 模板语法、`wx:if`、`wx:for`、事件绑定和数据绑定是否与项目一致
- 样式单位、条件编译、跨端差异是否有验证计划
- 变更后至少运行项目已有的 lint、类型检查、构建或目标平台编译命令
