# 西谷智灯暗涌系统 — 产品策略文档

> 西谷AI (Xigu AI) 的本土化编码 Agent，基于 VoltUI (Go + Wails) fork。

## 品牌名

- 用户面向名: **西谷智灯暗涌系统**
- 代码名: `voltui`（与上游一致，不改）
- 环境变量: `VOLTUI_BRAND_NAME=西谷智灯暗涌系统`

## 定位

西谷智灯暗涌系统是面向中国开发者团队的 AI 编码 Agent，核心价值：

1. **本地运行**: 无需云端，数据不出境，企业内网友好
2. **多模型**: DeepSeek / OpenAI / 本地 Ollama 自由切换
3. **Go 性能**: 比 Electron 类产品内存低 10x，启动快 5x
4. **MCP 协议**: 内置 MCP 服务器，可扩展工具链
5. **本土化**: 中文交互优先、CNB 云构建、中国合规适配

## Fork 筡理原则

- **源码与上游一致**: 仅通过 BrandConfig 定制品牌，不改源码
- **CI/CD 自定义**: `.cnb.yml` 添加自动发版 + MR 门禁
- **向上游贡献**: 通用功能先提 PR 到 volt-gui，再在 fork 享受
- **行业 skill**: fork 专属 skill (anyong-brand-config, cnb-ci-cd, xigu-ai-ops) 不贡献上游

## 发布管道

```
开发者 → feat: 提交 → CNB CI (版本号) → desktop-v* tag → GitHub Actions → 原生 runner 构建 → Release
```

- macOS: `.zip` + `.dmg` (universal)
- Windows: `-installer.exe` (NSIS)
- Linux: `.tar.gz` (amd64)

## 目标用户

1. **个人开发者**: 本地 AI 编码助手，DeepSeek 模型低成本
2. **企业内网**: 数据不出境，私有模型部署
3. **开发团队**: MCP 工具链共享，团队编码规范自动化
